Para executar o programa:

Compila o código:
`g++ main.cpp -o programa3d -framework OpenGL -framework GLUT`

Executar o programa:
`./programa3d`

Controles do Programa

- W - Avançar
- S - Recuar
- A - Rotacionar esquerda
- D - Rotacionar direita
- Q - Subir
- E - Descer
- Espaço - Ligar/desligar animação da bola
- ESC - Sair
